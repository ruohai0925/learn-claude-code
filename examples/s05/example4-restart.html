### How to Add More Tools

Just add a decorated function to `server.py`:
```python
@mcp.tool()
def my_tool(param: str) -> str:
    """One-line description shown to Claude."""
    return f"Result: {param}"
```
No other registration needed -- `FastMCP` handles the rest automatically.
